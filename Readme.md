# Руководство по работе с проектом

### Запуск проекта

```bash
npm start
```


### Запуск тестов

```bash
npm test
```


### Проверка линтером

```bash
npm run eslint
```


### Сборка проекта

```bash
npm run build
```


### Извлечение конфигурации проекта

```bash
npm run eject
```
